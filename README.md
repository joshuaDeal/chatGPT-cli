# ChatGPT-cli
A simple python script for interfacing with ChatGPT in the command prompt.

## Usage
```chatgpt-cli.py --help```

```
chatgpt-cli.py
Usage:  ./chatgpt-cli.py --[option]
Options:
        --help                  Display this help message.
        --key <keyfile>         Specify an api key file.
```
